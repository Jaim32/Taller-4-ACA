# Bienvenido a la Documentación de QGIS

**QGIS** (Quantum GIS) es un Sistema de Información Geográfica (SIG) de código abierto que permite visualizar, editar y analizar datos geoespaciales.

![Logo QGIS](img/logo.png)

## ¿Qué puedes hacer con QGIS?

- Visualizar mapas y capas vectoriales o raster
- Realizar análisis espaciales
- Conectarte a bases de datos como **PostGIS**
- Publicar mapas en la web
- Automatizar tareas con **Python (PyQGIS)**
